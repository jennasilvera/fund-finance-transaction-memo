# Project Summary

## Fund Finance Transaction Memo & Credit Request Package

This project is a simulated analyst-style fund finance credit package for a Corporate & Leveraged Finance – Fund Solutions Group Analyst role.

The project models a $150.0 million senior secured hybrid subscription / NAV revolving credit facility for a simulated private credit fund. It includes synthetic fund, investor, NAV, borrowing base, covenant, and monitoring data.

## What This Project Demonstrates

- Fund finance transaction analysis
- Credit request preparation
- Borrowing base analysis
- NAV / LTV analysis
- Facility availability calculation
- Investor concentration testing
- Covenant monitoring
- Closing memo preparation
- Term sheet support
- Lender presentation support
- SQL-based portfolio monitoring
- Excel model generation
- Python automation using pandas

## Core Outputs

| Output | Description |
|---|---|
| Transaction Memorandum | Internal-style transaction memo and credit narrative |
| Credit Request Package | Approval request with facility terms, strengths, risks, and recommendation |
| Excel Model | Fund finance model with assumptions, borrowing base, NAV, facility summary, and monitoring tabs |
| Term Sheet | Indicative simulated term sheet |
| Closing Memo | Documentation review support and conditions precedent checklist |
| Portfolio Monitoring Report | Monthly monitoring summary with covenant status and escalation triggers |
| SQL Monitoring Report | SQL-generated monitoring outputs from a SQLite database |
| Lender Presentation Outline | Mini pitch deck outline for lender or internal committee presentation |

## Important Disclaimer

All data is synthetic. This project does not use real clients, real funds, real sponsors, real investors, real bank information, real legal documents, or proprietary transaction data.
